HTML

HTML son las siglas de HyperText Markup Language . Es el lenguaje de marcado principal para la creación de páginas web.

Creado por Tim Berners-Lee, HTML ha evolucionado significatvamente desde sus inicios

Historia de HTML

Los inicio del lenguaje HTML se remontan a principios de la decada de 1990. Desde entoncs, ha pasado por varias versiones. cada una añadiendo nuevas funcionalidades y mejoras.

Versiones de HTML

Tim Berners-Lee definió la primera versión de HTML en el año 1991.

En la actualidad, la última versión de HTML es HTML5.
